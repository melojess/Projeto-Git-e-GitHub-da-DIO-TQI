Anotações para uso do GIT


O computador utilizado é o MAcBook M1 com silicon

Para a instalação do Git foi preciso utilizar o gerenciador de pacotes Homebrew

Para a resolução do problema foi utilizado os seguintes comandos 

cd /oopt/homebrew/bin

PATH=$PATH:/opt/homebrew/bin 

cd 

touch . zshrc 

echo export PATH=$PATH:/opt/homebrew/bin >> .zshrc
